Personal finance management application built with MERN stack.

## Features

- **Add Transactions** - Record income and expenses with descriptions
- **Real-time Balance** - Automatic calculation of current balance
- **Monthly Statistics** - View income, expenses, and transaction count for current month
- **Secure Authentication** - User registration and login system

## How to Run the Project

git clone .https link from the top.

cd budget-tracker

npm run dev
